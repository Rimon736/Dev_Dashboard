Question-1: What are the different ways to select an element in the DOM?
  1. getElementById("id")
  2. getElementsByClassName("class")
  3. getElementsByTagName("tag")
  4. querySelector("css-selector")
  5. querySelectorAll("css-selector")
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML allows us to get or set the HTML inside an element, meaning it processes and renders any tags included. innerText only deals with the visible text, ignoring hidden elements and formatting the text as it appears on the screen. textContent retrieves or sets all text inside an element, including hidden elements, but does not parse HTML. While innerHTML is useful for injecting HTML content, innerText is better for working with visible text, and textContent is the most efficient for handling raw text.
Question-3: What is event delegation in the DOM?
GEvent delegation is a technique in the DOM where a parent element handles events for its child elements, even if they are added dynamically. Instead of attaching event listeners to multiple child elements, we add a single listener to a common ancestor. This works because events bubble up from the target element to its ancestors. For example, adding a click event to a -ul- and detecting clicks on its -li- items is more efficient than attaching separate listeners to each -li- This improves performance and keeps code cleaner, especially when working with dynamic content.
Question-4: What is event bubbling in the DOM?
Event bubbling in the DOM is a process where an event starts from the target element and propagates upwards through its parent elements. When an event occurs on a child element, it first triggers on that element, then moves up to its ancestors in order until it reaches the root (document). For example, if we click a button inside a -div-, the click event first happens on the button, then bubbles up to the -div-, then to -body-, and so on. This behavior allows us to use event delegation and manage events efficiently at a higher level.
Question-5: How do you create, add, and remove elements using JavaScript?
Creating an element: Use document.createElement("tag") to create a new element.
Adding an element: Use parent.appendChild(child) to add an element at the end. Use parent.append(child) (modern alternative). Use parent.prepend(child) to add at the beginning.
Removing an element: Use element.remove() (modern way). Use parent.removeChild(child) (older method).
